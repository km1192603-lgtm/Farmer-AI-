# MVP Scope (8-10 Weeks)

## Must-Have (Phase 1)

1. Farmer onboarding profile
   - Language preference (Tamil, English, Telugu, Malayalam, Kannada)
   - District, land size, water availability, crop type
2. Unified AI assistant (text first)
   - Ask problem and get practical crop guidance
   - Context-aware answers based on farmer profile
3. Pest and disease basic diagnosis
   - Image upload
   - Crop + symptom input
   - Remedy and prevention output with confidence note
4. Weather and risk alerts (v1)
   - Daily forecast
   - Crop-stage risk flags and action recommendations
5. Fresher farmer roadmap planner
   - Seasonal plan from land prep to harvest
   - Budget estimator using seed/fertilizer and basic labor assumptions

## Should-Have (Phase 2)

1. Voice assistant (STT/TTS)
2. Fertilizer recommendation with NPK split plan
3. Government schemes finder (basic eligibility)

## Could-Have (Phase 3)

1. Market price trend and prediction
2. Proactive notification engine (SMS/WhatsApp)
3. Admin content override tools

## Out of Scope for MVP

1. Marketplace/e-commerce transactions
2. Insurance underwriting workflows
3. Full offline application support (only draft caching in MVP)

## Priority Matrix

- P0: Onboarding, text assistant, pest image diagnosis, weather alerts, fresher planner
- P1: Voice assistant, fertilizer planner, schemes finder
- P2: Market prediction and advanced automation
