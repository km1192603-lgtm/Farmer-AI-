# Pilot Selection and Language Rollout

## Pilot Geography (India)

1. Coimbatore, Tamil Nadu
2. Guntur, Andhra Pradesh

These districts are selected for:
- Strong agricultural activity
- Diverse crop patterns
- Good baseline mobile usage

## Pilot Crops

1. Paddy (Rice)
2. Tomato
3. Cotton

## Language Rollout Strategy

### Wave 1 (MVP)
- Tamil
- English

### Wave 2
- Telugu
- Kannada

### Wave 3
- Malayalam

## Acceptance Targets for Pilot Exit

- 500+ onboarded farmers
- 60%+ weekly active usage
- 70%+ successful pest diagnosis submissions
- 50%+ roadmap planner completion rate
